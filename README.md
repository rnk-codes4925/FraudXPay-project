# FraudXPay - Fraud Detection App

A Machine Learning app to detect fraudulent transactions.

## Live App
🚀 [Click here to open app](https://fraudxpay-project-gpz7appxyyjdsgivrdpdki3.streamlit.app)

## Features
- Detects fraud in bank transactions
- Built with Python, Scikit-learn and Streamlit
- Trained on 6 million+ real transactions
